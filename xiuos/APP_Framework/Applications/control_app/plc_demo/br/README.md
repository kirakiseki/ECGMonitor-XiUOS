# 贝加莱 PPC2100通信测试

[TOC]

## 通信接线及参数设置

* 网口
  * 通过ETH1 RJ45 网口连接
  * 网口参数：IP：192.168.250.26  Port：502
  * 测试的协议:Modbus TCP

## 存储区

- 贝加莱PLC与其他PLC不同，没有明确类似M，D等这样的存储区的概念，Modbus地址取决于库函数中结构体变量索引。

## 通信测试

-  共测试BOOL，INT16，INT32，FLOAT,DOUBLE 共五种类型数据。
-  测试BOOL型变量，用功能码01，测其他类型变量用功能码03。
